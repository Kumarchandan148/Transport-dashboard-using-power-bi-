# Transport and Delivery Analysis dashboard-using-power-bi
This project presents a Power BI dashboard designed to analyze transportation and delivery performance across multiple states. It visualizes key logistics metrics such as Total Miles, Revenue, Shipping Cost, Trip Types, and Profit to support data-driven planning and decision-making.


Purpose of the Project

To analyze transportation efficiency across states

To compare revenue, cost, and miles traveled

To identify which states contribute the most to deliveries

To monitor trip types (Domestic, International, Intercom)

To assist logistics decision-makers in improving delivery performance

To make data-driven decisions for route optimization and cost reduction
Technology Uesd:-
Power Bi desktop
DAX Query
Visualization tool
